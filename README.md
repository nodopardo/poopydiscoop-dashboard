# Poopydiscoop Dashboard (Streamlit)

## Archivos
- app.py
- requirements.txt
- Poopydiscoop.xlsx

## Deploy (Streamlit Community Cloud)
1) Sube estos 3 archivos al repo (GitHub).
2) En Streamlit: New app -> selecciona el repo -> Main file: app.py
3) Deploy.

Si aparece error:
- Revisa que el archivo se llame EXACTAMENTE `Poopydiscoop.xlsx`.
- Revisa que el Excel tenga columna `Miembro` (o que la primera columna sea el nombre de cada persona).
- Revisa que existan las hojas (por ejemplo: `Shit 2024`, `Shit 2025`).
